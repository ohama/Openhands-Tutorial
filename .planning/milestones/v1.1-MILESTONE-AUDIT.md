### TD-3 — Off-by-one in event count  *[Documentation]*

**Fix when:** anytime; trivial.

### TD-4 — Cosmetic mdbook WARN  *[Cosmetic]*

`mdbook build` emits one WARN: `unclosed HTML tag <char> found in appendix-c-comparison.md while exiting TableCell`. Caused by `LexBuffer<char>.FromString` inside a backtick code span within a table cell — mdbook's HTML parser doesn't see the backticks. Build exits 0; HTML renders correctly.

**Fix when:** optional; could escape `<char>` to `&lt;char&gt;` or restructure that table cell.

### TD-5 — Bibliography paths in Sources section  *[Reader UX]*

The Sources section in `appendix-c-comparison.md` lists `.planning/` relative paths as references. These are not clickable mdbook links — they're audit-trail bibliography for the published `.planning/` directory. Not breakage, but a reader who wants to dig deeper has no clickable affordance.

**Fix when:** v1.2 polish pass if priorities call for it; could replace with linked GitHub URLs into the `.planning/` tree.
