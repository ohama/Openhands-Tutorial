<execution_warning>
**THIS PLAN IS SLOW / EMPIRICAL / --yolo.**
- Each `openhands` invocation is a long autonomous agent run on a LOCAL 122B model (~25-30% slower than 35B). Budget ~6-8 minutes per complex task, up to ~15 minutes if the agent gets stuck debugging the lexer. The full run may take 30-60+ minutes.
- The CLI exits 0 REGARDLESS of task success — DO NOT judge outcome from exit code. Judge ONLY from the JSONL (FinishAction present, exit codes inside TerminalObservations, actual command output).
- Run each invocation in the BACKGROUND and POLL the JSONL until the run settles (FinishAction event appears, or no new events for a sustained period, or ConversationErrorEvent). Do not block the foreground on `sleep`.
- Outcomes are NOT pre-determined. The agent may write a working lexer, may hit build errors and self-correct (the desired error-and-fix), may need a retry, or may need the disclosed fallback. Follow the decision rules below and record what ACTUALLY happens.
</execution_warning>
