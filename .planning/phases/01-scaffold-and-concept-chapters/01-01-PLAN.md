This is the ONE deliberate exception to "no stub files" — it applies only to the seven Phase-1 chapter files that WILL be written this same phase, never to Phase 2-5 chapters (those stay as draft entries with empty hrefs).

Purpose: Establishes the URL-stable book structure so later chapters slot in without reordering or breaking GitHub Pages links.
