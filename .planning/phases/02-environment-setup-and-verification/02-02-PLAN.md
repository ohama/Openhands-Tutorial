Notes:
- --json only emits JSONL WITH --headless. --override-with-envs is REQUIRED or env vars are ignored.
- --yolo auto-approves tool calls (user-authorized).
- Use a dedicated scratch working dir so the --yolo agent stays bounded.
- Each line of stdout is one JSON event. ActionEvent has an `action` (tool call); ObservationEvent
  has an `observation` (tool output). Success = an action running the echo, and an observation whose
  output contains the sentinel.

  3. Parse the JSONL to confirm an action+observation pair:
     ```
     python3 -c "
     import json,sys
     acts=obs=0; hit=False
     for line in open('/Users/ohama/projs/OpenHandsTests/oh-workdir/ping.jsonl'):
         line=line.strip()
         if not line: continue
         try: ev=json.loads(line)
         except: continue
         s=json.dumps(ev)
         if 'OPENHANDS_PING_OK' in s and ('echo' in s or 'command' in s): pass
         if ev.get('action'): acts+=1
         if ev.get('observation'): obs+=1
         if 'OPENHANDS_PING_OK' in s and ev.get('observation'): hit=True
     print('actions=',acts,'observations=',obs,'sentinel_in_observation=',hit)
     assert acts>=1 and obs>=1 and hit, 'PING FAILED'
     print('PING PASS')
     "
     ```

  - If the LLM call is refused / connection error to 127.0.0.1:4000: confirm 02-01 proxy check;
