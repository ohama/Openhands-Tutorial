# 이 튜토리얼에 대하여

## 목적

이 튜토리얼은 에이전틱 AI(Agentic AI)를 이론이 아닌 실제 동작으로 배우는 것을 목표로 합니다.
OpenHands라는 오픈소스 AI 에이전트 플랫폼을 사용하여, LLM이 스스로 계획을 세우고, 코드를 작성하고, 빌드하고, 테스트하는 전 과정을 직접 눈으로 확인합니다.
단순히 "AI가 코드를 생성한다"는 수준을 넘어, 에이전트가 어떻게 판단하고 행동하는지 그 구조와 흐름을 이해하는 것이 핵심입니다.

## 독자

기본적인 프로그래밍 경험이 있는 분을 대상으로 합니다.
F#이나 함수형 프로그래밍 지식은 필요하지 않습니다. 에이전트가 F# 계산기를 처음부터 작성하는 과정을 보여주므로, 독자는 코드의 세부 내용보다 에이전트의 동작 방식에 집중하면 됩니다.
OpenHands 자체를 개발하거나 커스터마이징하려는 분보다는, 에이전틱 AI가 실제로 어떻게 작동하는지 이해하고 싶은 분에게 적합합니다.

## 선행 지식

다음 정도의 배경 지식이 있으면 충분합니다:

- 터미널(명령행)을 기본적으로 사용할 수 있는 수준
- Docker의 개념을 알고 있거나 간단히 실행해본 경험
- 프로그래밍 언어 하나 이상을 읽고 이해할 수 있는 수준

## 구성 및 소요 시간

이 튜토리얼은 총 5부로 구성되어 있습니다:

- **1부**: 에이전틱 AI의 개념과 핵심 용어를 소개합니다.
- **2부**: OpenHands의 아키텍처와 내부 동작 원리를 설명합니다.
- **3부**: OpenHands와 로컬 Qwen LLM 서버를 설치하고 연결하는 환경 설정 과정을 안내합니다.
- **4부**: OpenHands가 F# 계산기를 자율적으로 만드는 전 과정을 단계별로 살펴봅니다.
- **5부**: 배운 내용을 정리하고 다음 단계를 안내합니다.

**성능 안내:** 이 튜토리얼의 실행 캡처는 로컬 Qwen 35B 모델(http://127.0.0.1:8000/v1)을 사용했습니다.
요청당 응답 시간이 약 240초(4분)에 달하므로, 여러 단계를 거치는 에이전트 실행은 상당한 시간이 소요됩니다.
이것은 성능 문제가 아니라, 대형 로컬 모델의 현실적인 특성입니다. OpenHands Cloud나 더 빠른 API를 사용하면 속도가 크게 단축됩니다.
